This extension provides messaging and document replication functions for the eXist XML database.
  • The messaging xquery extension functions alows to send and asynchronously receive JMS messages to/from a message broker.
  • The document replication feature asynchronously copies and transports documents to other eXist-db instances.
Both features use the Java Messaging Services standard to send and receive data.

Register Receiver

jms:register( $callback as function(*), $additionalParameters as item()*, $jmsConfiguration as map(*) )
